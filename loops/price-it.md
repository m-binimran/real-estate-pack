# /price-it $ARGUMENTS

Produce a CMA for the property in `$ARGUMENTS` using the `cma` skill. Evidence at every step.

1. **Profile the subject** (location, sqft, beds/baths, lot, condition, age, features).
2. **Gather real comps** - recent solds + active/pending competition from the MLS/public records. Never
   invent addresses or prices; mark any data the agent must pull `[verify]`.
3. **Adjust** comps to the subject for material differences; keep adjustments consistent and conservative.
4. **Reconcile** to a supported price RANGE, weighting the closest comps.
5. **Explain** the range and the market trend in plain English for the client.

Output: comps table + range + rationale, clearly labeled an **opinion of value for pricing, not an
appraisal**, with `[verify]` on any unconfirmed comp data.
