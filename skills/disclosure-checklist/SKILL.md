# disclosure-checklist

Disclosures vary heavily by state - this builds the checklist and flags what to confirm locally.

## Common items (confirm against YOUR state + MLS)
- **Federal:** Lead-Based Paint disclosure + EPA pamphlet for homes built before 1978.
- **Seller property condition / TDS-type disclosure:** known material defects, systems, history.
- **Natural hazard / environmental:** flood zone, wildfire, earthquake, etc. (state-dependent).
- **Material facts:** known issues affecting value/safety (deaths/stigma rules vary by state).
- **HOA documents** if applicable: CC&Rs, budget, fees, minutes.
- **Agency disclosure** and any **AI/photo** disclosure (virtual staging labeled).
- **Local/state-specific** addenda (e.g. CA TDS/NHD; transfer tax forms).

## Process
- Build the list for this property, mark each delivered/acknowledged, and **flag every item that depends on
  state law** with "confirm with broker/state requirements."

## Output
- A disclosure checklist: item -> required? -> delivered/acknowledged -> notes, with state-specific items flagged.

## Guardrails
- This is a **starting checklist, not legal advice** - your state, MLS, and brokerage define the binding
  requirements. Confirm with your broker / attorney.
- Never advise concealing a known material defect - disclose.
